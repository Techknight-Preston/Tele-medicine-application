## Telemedicine Android App

This project is an Android-based telemedicine application designed to help patinets book appointments with doctors and allow doctors to manage those appointments.

## Features
-Patient and doctor authentication
-Appointment booking and viewing System
-Records System
-View PatientRequests System
- E-prescription
- Find doctors system
- Send Inquiry System
- View Specialists system

- ## Technologies Used
- Java
- XML
- SQLite
- Android Studio

  ## Setup
  1. Clone this repository
  2. Open the project in Android Studio
  3. Let Gradle Sync
  4. Run the app on an emulator or a physical device
